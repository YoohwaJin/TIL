# linux_command

> 리눅스 명령어를 정리해봅시다.

- `pwd` (print working directory)
    - 현재 작업중인 경로를 출력

- `ls` (list)
    - 현재 폴더에 있는 파일, 폴더를 출력
    - `-a` (optional): 숨김처리된 파일, 폴더까지 출력

- `cd` (change directory)
    - 폴더 이동
    - 'cd..' 상위 파일로 이동
    - 'cd../ 파일명' 해당 파일로 이동
    ** tap버튼 자동완성 기능 십분 활용

- `mkdir` (make directory)
    - 폴더 생성

- `touch` 
    - 파일 생성

- `rm` (remove)
    - 파일 및 폴더 삭제
    - '-r'(optional): 폴더 삭제를 위해 입력해야하는 옵션
